# 🧪 Тестирование BookStoreWarehouse

## 📋 План тестирования

Проект готов к написанию тестов.

### Unit Tests (Модульные тесты)

**Инструменты:**
- xUnit или NUnit
- Moq (для моков)
- FluentAssertions

### Integration Tests (Интеграционные тесты)

**Инструменты:**
- xUnit
- TestContainers (для БД)
- HttpClient

### CI/CD Pipeline

GitHub Actions для автоматизации тестов при push/PR

**Happy testing! 🧪✅**
